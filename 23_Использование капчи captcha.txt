            1. Теперь добавим КАПЧУ

        1.1 установим django-simple-captcha
            pip install django-simple-captcha
        1.2. как привязать капчу к нашему сайту найдем по ссылке https://django-simple-captcha.readthedocs.io/en/latest/usage.html#installation
        1.3. в settings.py в INSTALLED_APPS добавляем 'captcha',

        1.4. сделаем миграцию
                python manage.py migrate

        1.5. добавим глобальный путь в coolsite/urls.py
                urlpatterns += [path('captcha/', include('captcha.urls')),]

            2. Используем капчу в нашей форме : https://django-simple-captcha.readthedocs.io/en/latest/usage.html#adding-to-a-form
        2.1. импортируем в forms.py
            from captcha.fields import CaptchaField
        2.2. в нашей форме "class ContactForm(forms.Form)" добавим
            capatcha = CaptchaField()
        2.3. запускаем сервер и видим капчу
    ее отображение мы можем изменить стандартными методами
